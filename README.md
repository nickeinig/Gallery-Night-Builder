# Gallery Night

Standalone local HTML builder for LJC Gallery Night marketing boards.

## Files

- `Gallery Night Studio.html`: live builder used to assemble office posters.
- `assets/`: local fonts, intern headshots, and support files used by the builder.
- `Working Files/options/`: poster style explorations and alternate directions.

## Use

Open `Gallery Night Studio.html` in a browser, choose an office, add projects, and export the final board as PNG or PDF.
